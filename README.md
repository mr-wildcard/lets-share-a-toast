# 🍞 Let's share a TOAST 🍞

## Installation

1. `pnpm i`
2. `unzip firebase/firestore-data.zip`

## Getting started

```
$ pnpm dev
```

- Webapp URL: [http://localhost:5000/](http://localhost:5000/)
- Firebase emulator suite: [http://localhost:5004/](http://localhost:5004/)

## Upgrade project dependencies

```
$ pnpm up -r -i --latest
```
